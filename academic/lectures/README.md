## How to Use

Each lecture contains:
- **Overview** — what the lecture covers and why
- **Key Concepts** — deep explanations with formal definitions
- **Formal Definitions/Models** — mathematical formulations
- **Worked Examples** — step-by-step problem solving
- **Arena Connection** — how theory maps to the TendedLoop Arena platform
- **Discussion Questions** — for class participation and deeper thinking
- **Further Reading** — beyond the required textbooks

See the [Academic Program](../README.md) for syllabi, assessment, and course integration options.
